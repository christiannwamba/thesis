About this initiative

Overview

This application was an initiative that is powered by Nwamba Christian and Engr. Okpalah Etomchi of Computer Engieering Department, Caritas University.

It's objective was not just to serve as a thesis backup but to serve as a contribution in making Nigeria's academic strategy more interesting and robust

It satisfies all cliams made by Christian Nwamba to serve for his thesis but is still open for contribution and source code is avalable on GitHub

How to

Using this application is quite simple:

  1. Create an account and login. This generates an id for you and will serve as your call number
  2. The lecturer would provide his id for each lecture and you can join the class by calling hid id
